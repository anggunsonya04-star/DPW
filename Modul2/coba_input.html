Uji Coba 2



Kesimpulan: ketika atribut "name" tidak diberikan, semua pilihan radio button dapat dipilih secara bersamaan karena browser menganggap mereka tidak saling berhubungan. Ketika atribut "name" diberikan dengan nilai yang sama pada semua radio button (seperti name="pilihan"), hanya satu pilihan yang bisa dipilih dalam satu waktu — pilihan baru akan otomatis membatalkan pilihan sebelumnya. Ketika atribut "name" diberikan dengan nilai yang berbeda pada setiap radio button, masing-masing berdiri sendiri sehingga semua bisa dipilih bersamaan, sama seperti tidak diberi name. Sehingga atribut "name" yang sama pada radio button berfungsi untuk mengelompokkan pilihan agar hanya satu yang bisa dipilih dalam satu grup.


Uji Coba 3



Kesimpulan: Checkbox memungkinkan pengguna untuk memilih lebih dari satu pilihan sekaligus. Atribut "name" pada checkbox tidak membatasi jumlah pilihan — baik diberi name yang sama maupun berbeda, semua checkbox tetap bisa dicentang bersamaan. Atribut "name" pada checkbox berfungsi untuk pengelompokan data saat dikirim ke server, bukan untuk membatasi pilihan.


Uji coba 4




Kesimpulan: Input type="color" menampilkan color picker sehingga pengguna dapat memilih warna dengan mudah secara visual. Input type="date" menampilkan kalender untuk memilih tanggal, sehingga format tanggal selalu konsisten. Input type="button" menampilkan tombol yang bisa menjalankan fungsi JavaScript melalui atribut onclick, sangat berguna untuk interaksi tanpa harus mengirim form.


Uji Coba 5




Kesimpulan: Input type="datetime-local" memungkinkan pengguna memilih tanggal sekaligus waktu (jam dan menit). Tampilan dapat berbeda antar browser. Input type="month" memungkinkan memilih bulan dan tahun saja tanpa tanggal spesifik. Input type="email" secara otomatis memvalidasi format email — browser akan menolak input yang tidak mengandung karakter "@" saat form disubmit. Tampilan di beberapa browser lama mungkin sama seperti input teks biasa.


Uji Coba 6




Kesimpulan: Input type="image" menampilkan gambar yang berfungsi sebagai tombol submit, berguna untuk membuat tombol kirim yang lebih menarik secara visual. Input type="file" menampilkan tombol untuk memilih file dari perangkat pengguna, cocok untuk fitur upload foto atau dokumen. Input type="hidden" tidak ditampilkan kepada pengguna sama sekali, namun nilainya tetap dikirim ke server saat form disubmit. Berguna untuk menyimpan data seperti ID pengguna atau token keamanan secara tersembunyi.


Uji Coba 7



Kesimpulan: Input type="number" menampilkan kotak angka dengan tombol naik/turun. Atribut min dan max membatasi nilai yang bisa diinput. Ketika nilai min diubah menjadi -50, slider/angka akan dimulai dari -50 sehingga pengguna dapat memilih nilai negatif. Input type="range" menampilkan slider yang lebih intuitif untuk memilih nilai dalam rentang tertentu, namun tidak menampilkan nilai secara langsung kepada pengguna./p>

Uji Coba 8




Kesimpulan: Input type="url" memvalidasi bahwa nilai yang dimasukkan merupakan format URL yang valid (harus diawali https:// atau http://). Browser akan menampilkan peringatan jika format salah. Input type="search" tampilannya mirip input teks biasa namun beberapa browser menambahkan tombol "X" untuk menghapus teks dengan cepat. Input type="tel" tidak memvalidasi format nomor telepon secara otomatis di semua browser. Atribut "pattern" dapat digunakan untuk menentukan format yang valid. Di perangkat mobile, input ini akan menampilkan keyboard angka.


Uji Coba 9



Kesimpulan: Input type="time" menampilkan picker untuk memilih jam dan menit. Di beberapa browser lama atau Firefox, tampilannya mungkin berupa input teks biasa. Input type="week" memungkinkan pengguna memilih minggu ke berapa dalam satu tahun. Dukungan antar browser tidak seragam — Chrome mendukung dengan baik, namun Firefox dan Safari mungkin menampilkannya sebagai input teks biasa. Kesimpulan umum: Tidak semua tipe input HTML5 didukung secara penuh oleh semua browser. Browser modern seperti Chrome cenderung mendukung lebih banyak tipe input dibandingkan browser lama.
