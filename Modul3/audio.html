    Kesimpulan Uji Coba Audio:

    Tag <audio> digunakan untuk menyematkan file suara ke dalam halaman web tanpa memerlukan plugin tambahan. File audio didefinisikan menggunakan tag <source> dengan atribut src (lokasi file) dan type (format file).

    Atribut controls → menampilkan panel kontrol audio bawaan browser seperti tombol play/pause, pengatur volume, dan progress bar. Tanpa atribut ini, audio tidak memiliki tampilan apapun di halaman.

    Atribut autoplay → menyebabkan audio langsung diputar otomatis ketika halaman selesai dimuat. Namun di browser modern seperti Chrome dan Firefox, autoplay akan diblokir jika tidak disertai atribut muted karena kebijakan browser untuk mencegah gangguan kepada pengguna.

    Atribut muted → membuat audio diputar dalam kondisi bisu (tidak bersuara). Jika dikombinasikan dengan autoplay, audio akan langsung diputar otomatis namun tanpa suara. Pengguna masih bisa mengaktifkan suara secara manual melalui panel kontrol.
