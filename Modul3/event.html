    • Mouse Event pada Gambar

    Arahkan mouse atau klik pada gambar untuk melihat event!

    Kesimpulan Mouse Event:
    onclick → event dipicu ketika gambar diklik satu kali.
    ondblclick → event dipicu ketika gambar diklik dua kali berturut-turut.
    onmouseover → event dipicu ketika kursor mouse masuk ke area elemen (termasuk elemen anak).
    onmouseout → event dipicu ketika kursor mouse keluar dari area elemen.
    onmouseenter → mirip onmouseover namun tidak dipicu oleh elemen anak di dalamnya.
    onmouseleave → mirip onmouseout namun tidak dipicu oleh elemen anak di dalamnya.
    onmousedown → event dipicu ketika tombol mouse ditekan (belum dilepas).
    onmouseup → event dipicu ketika tombol mouse yang ditekan dilepaskan.
    onmousemove → event dipicu terus-menerus selama kursor mouse bergerak di atas elemen.
    Mouse event sangat berguna untuk membuat halaman web lebih interaktif tanpa harus berpindah halaman.
