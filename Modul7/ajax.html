Nomor 8: Uraian AJAX dan jQuery

1. Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah teknik pemrograman web yang memungkinkan halaman web berkomunikasi dengan server tanpa harus me-reload seluruh halaman. Data dikirim dan diterima secara asynchronous di latar belakang. Cara Kerja AJAX: - User memicu event (klik tombol, ketik teks, dsb.) - JavaScript membuat objek XMLHttpRequest - Request dikirim ke server secara asynchronous - Server memproses dan mengirimkan response (JSON/XML/HTML) - JavaScript menerima response dan memperbarui DOM tanpa reload

2. Apa itu jQuery?

jQuery adalah library JavaScript yang mempermudah manipulasi DOM, penanganan event, animasi, dan penggunaan AJAX. Prinsipnya: "Write less, do more".

3. Cara Menggunakan jQuery

Untuk menggunakan jQuery, Anda perlu memasukkan library jQuery ke dalam file HTML Anda, baik dengan mengunduh file-nya secara lokal atau menggunakan Content Delivery Network (CDN) seperti yang dilakukan pada kode ini.
